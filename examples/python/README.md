# Python mTLS Examples

This directory contains Python scripts demonstrating how to use the generated mTLS certificates.

## Environment Setup

To run these examples, you need to set up the Python environment at the **project root**:

```bash
# Navigate to project root
cd ../../

# Create a virtual environment
python3 -m venv .venv

# Activate the virtual environment
source .venv/bin/activate  # On Linux/macOS
# or
.venv\Scripts\activate     # On Windows

# Install required packages
pip install -r requirements.txt
```
